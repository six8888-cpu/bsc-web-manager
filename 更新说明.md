## 🚀 快速开始

### 方法1：从GitHub安装（推荐）

```bash
# 完整安装（包含域名SSL配置选项）
bash <(curl -sL https://raw.githubusercontent.com/YOUR_USERNAME/bsc-web-manager/main/deploy/install_all.sh)
```

### 方法2：手动克隆安装

```bash
# 1. 克隆项目
git clone https://github.com/YOUR_USERNAME/bsc-web-manager.git

# 2. 进入目录
cd bsc-web-manager

# 3. 执行安装
sudo bash deploy/install_all.sh
```

## 📋 安装流程

### 步骤1: 系统检测
- 自动检测操作系统
- 显示服务器IP
- 检查root权限

### 步骤2: 安装环境
- 更新系统
- 安装Python、Git等依赖
- 创建虚拟环境
- 安装Python包

### 步骤3: 配置防火墙
- 开放5000端口（Web服务）
- 开放80端口（HTTP）
- 开放443端口（HTTPS）

### 步骤4: 域名SSL配置（可选）
用户可选择：
1. ✅ 配置域名和SSL（推荐）
   - 输入域名
   - 输入邮箱
   - 自动申请证书
   - 配置Nginx
2. ⏭️ 跳过，使用IP访问

### 步骤5: 创建服务
- 创建systemd服务
- 设置开机自启
- 启动服务

## 🔐 SSL证书配置

### 自动配置包含

1. **Nginx安装和配置**
   - 反向代理设置
   - WebSocket支持
   - HTTP自动跳转HTTPS

2. **SSL证书申请**
   - Let's Encrypt免费证书
   - 自动验证域名
   - 90天有效期

3. **自动续期**
   - 每天凌晨3点检查
   - 自动续期即将过期的证书
   - 续期后自动重载Nginx

## 🛠️ 服务管理

### 基本命令
```bash
sudo systemctl start bsc-web-manager      # 启动
sudo systemctl stop bsc-web-manager       # 停止
sudo systemctl restart bsc-web-manager    # 重启
sudo systemctl status bsc-web-manager     # 状态
sudo journalctl -u bsc-web-manager -f     # 日志
```

### SSL证书管理
```bash
sudo certbot renew                        # 手动续期
sudo certbot certificates                 # 查看证书
sudo systemctl status certbot.timer       # 查看自动续期任务
```
